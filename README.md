## 产品定位

保联是一款面向大陆客户的香港保险全链路数字化服务平台，由持牌保险经纪公司运营，让客户在一个 App 上查看所有香港保险公司的保单，同时获取专业的保险知识和顾问服务。

## 原型说明

### C端（客户 App）
- 4个一级 Tab：首页 / 保单 / 发现 / 我的
- 完整二级页面：保单详情、保障缺口分析、文章详情、产品详情、FAQ详情、消息中心、汇率详情、登录注册、实名认证等
- 所有卡片、文章、产品均可点击交互

### B端（经纪管理系统）
- 双角色视图：经纪人端 / 公司管理端（顶部切换）
- 经纪人端：工作台、客户管理、保单管理、咨询线索、佣金查看
- 公司管理端：数据看板、全局管理、经纪人管理、业绩排名、佣金管理、内容CMS、产品管理
